Spark with Scala

Spark is an open source computing framework up to 100 times faster than Mapreduce and Scala is a general-purpose programming language providing support for functional programming.

Spark with Scala Course objectives

Some of the key skills you will gain upon completion of this program include:
  • Program in Scala
  • Understand Scala's approach to object-orientation
  • Master the use of functional programming techniques in Scala
  • Understand how to perform TDD (test-driven development) using Scala
  • Install Spark and implement Spark operations on Spark Shell
  • Understand the role of Spark RDD
  • Implement Spark applications on YARN (Hadoop)
  • Learn Spark Streaming API
    • More...

Prerequisite

  • • Attendees should be Java developers planning to develop Scala applications. Hadoop Map Reducing, Hadoop Architecture.
  • • A basic understanding of functional programming and object oriented programming will help. Knowledge of Scala will definitely be a plus, but is not mandatory.
