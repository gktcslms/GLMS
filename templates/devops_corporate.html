Ruby and DevOps

Ruby is a dynamic, reflective, object-oriented programming language. DevOps is a software engineering culture and practice that aims at unifying software development and software operation.

Ruby and DevOps Course objectives

Some of the key skills you will gain upon completion of this program include:
  • Configure and install Ruby
  • Learn the basics of the Ruby language.
  • Learn DevOps Concepts
  • Learn Infrastructure Management using Chef

Prerequisite

  • • The participants taking Ruby and ChefTraining course must be familiar with the general principles of object oriented programming (OOP).
  • • Familiarity with IT environments & server administration for at least 6 months would be beneficial.
