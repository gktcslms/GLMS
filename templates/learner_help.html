Features In Learner Dashboard.


Learner Dashboard - My Courses

After the learner login, learner will be redirected to Learner Dashboard. Here you can see your purchased and subscribed courses.

Course Preview 1

After clicking on preview button, learner can see course page which contains Course Description, Course Objective, Prerequisite, Requirements and Course Contents.

Course Preview 2

In course preview page course content is devided in modules where each module can contain Presentation, Video, Labsession, Assignment and Multiple References

Apply Promo Code

If you have a Promo Code, you can apply here.

Submit your assignment

If you want to submit an assignment then you can select and submit your assignment(.pdf/.doc file).

My Submitted Assignment

In this section you can see all your submitted assignments
