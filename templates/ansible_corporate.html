Ansible

Ansible is an open source automation platform. It is very, very simple to setup and yet powerful. Ansible can help you with configuration management, application deployment, task automation

Ansible Course objectives

Some of the key skills you will gain upon completion of this program include:
  • Install and setup Ansible
  • Learn to Execute ad-hoc commands against servers using Ansible
  • Learn the fundamentals of Ansible
  • Learn to Optimize Ansible playbooks to reduce playbook execution time.
  • Comprehend loops, blocks, strategies, and galaxies
  • Understand the use of Ansible in Cloud domain
  • Learn to Test and troubleshoot Ansible playbook execution.
  • Learn to write operational playbooks to check cluster status and perform a cluster restart.
  • Learn to write Ansible configuration playbooks to deploy a 3-tier web application
    • More...

Prerequisite

  • • Linux or Mac OS X computer, or access to at lease one Linux virtual machine for installing Ansible.
  • • Prior exposure to Linux administrator skills
  • • A terminal and ssh client for running Ansible against target hosts.
