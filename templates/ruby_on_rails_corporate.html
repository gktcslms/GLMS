Ruby On Rails

Ruby is a Object Oriented Scripting language which is quick to learn, easy to deploy and provides substantial functionality required for e-commerce. Ruby on Rails is an extremely productive web application framework written in Ruby.

Ruby On Rails Course objectives

Some of the key skills you will gain upon completion of this program include:
  • Learn to program in Ruby
  • Learn to install and configure Ruby and Rails
  • Understand the Model-View-Controller framework architecture
  • Learn to build database-driven applications with Ruby on Rails
  • Learn to work with templates
  • Learn to use Ajax in Ruby on Rails applications
  • Learn to test Ruby on Rails applications
    • More...

Prerequisite

  • • The participants must be familiar with the general principles of object oriented programming (OOP).
  • • Basic knowledge of HTML
  • • Experience in the server-side programming and SQL would be beneficial
